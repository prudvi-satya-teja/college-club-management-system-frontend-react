# AClub — Campus Club Management System

A web app I built to manage clubs and events at our college. Students can discover clubs, join them, register for events, and coordinators/admins can manage everything from one place.

## What it does

- Browse all clubs on campus and view their events
- Join clubs and get assigned a role (Member, Coordinator, Admin)
- Create and manage events with images, date, location, theme
- Register for events and give feedback/ratings after attending
- Super Admin can manage all users and clubs
- Role-based access — you only see what you're supposed to

## Tech Stack

**Frontend**
- React + Vite

**Backend**
- Spring Boot
- Spring Security + JWT + OAuth
- JPA + Hibernate
- MySQL
- Cloudinary (for image uploads)
